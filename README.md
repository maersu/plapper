# Plapper

## Description

This is an implementation of a simple text sharing application and make use of the [Parse JavaScript API](http://parse.com).

Live demo: http://plapper.parseapp.com/

## Commands

install:
  bower install

Start a local server:
  grunt serve

Build:
  grunt

Deploy:
  grunt deploy
